# Классификация комментариев

## Данные

Тексты комментариев, с разметкой об их токсичности

## Задача

Натренировать модель, классифицирующую комментарии на нейтральные и токсичные для последующей отправки последних на модерацию.

*numpy*

*pandas*

*tqdm*

*plotly*

*sklearn*

*tune_sklearn*

*lightgbm*

*torch*

*skorch*

*transformers*
